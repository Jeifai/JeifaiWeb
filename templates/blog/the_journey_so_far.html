The journey so far.

Rollercoaster


It might sounds obvious but let's repeat it: 2020 has been an unusual year from too many points of view.

Without taking into account the dramatic events occoured with the global pandemic emergency, Jeifai's develop has been an incredible rollercoaster till now.

The idea to start this project has came so naturally that from one day to another, the team was diving into coding at a pace never knew before.

As a proper rollercoaster would do, this year has fluctuated across moments of slow speed into moments at speed of lights.

Sometimes ideas were flourishing so fast, that was necessary to completely stop any development in order to figure them out properly.

An enourmous part of the work has been done, but now the rollearcoasted is facing new challenges.

Exciting times ahead!

Technologies


We are here to talk about technolgies.

Developing Jeifai has required to learn an amount of new tech stuff not predicatable at the beginning, and it has been a really beautiful journey.

First of all, the whole backend is written in Golang. The idea was to try something new, more solid than the standalone Python or Ruby.

It as been a fantastic idea, because thanks to Golang new levels of understanding have been reached, mostly from an architectural point of view.

Developing a medium size web application requires a good amount of pattern, and Golang has revelead as an optimal choice because its flexibility.

Then it came the frontend, and there has been a pretty long debate about which framework start to use.

VueJS has won the challenge, because of its freshness and low entry level barriers. Vue is a pure pleasure to use, particularry in moving data across different sections of the application.
